How to Use

This keyboard tester helps you check if all keys on your keyboard are working properly.

  • Press any key to see its information displayed above
  • The corresponding key on the visual keyboard will light up
  • Test modifier keys (Shift, Ctrl, Alt) by holding them down
  • Check special keys like Function keys, Arrow keys, etc.
